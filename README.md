# 清水8組　回覧順序ファイル

## 📋 概要

清水8組の回覧板用の順序表です。  
A4縦1枚に3部印刷できるフォーマットになっています。  
係の方が毎年・毎回使いまわせるよう、**名前の記入欄が空白**になっています。

---

## 📥 ファイルのダウンロード方法

1. このページ上部の緑色のボタン **「<> Code」** をクリック
2. **「Download ZIP」** を選択してダウンロード
3. ZIPを解凍すると `清水8組_回覧順序v4.docx` が入っています

または、ファイル名をクリックして開き、右上の **「↓ ダウンロード」ボタン** からも取得できます。

---

## ✏️ 使い方

1. `清水8組_回覧順序v4.docx` をダウンロードして開く（Microsoft Word 推奨）
2. 各セルに**縦書きで名前を入力**してください
3. 入力が終わったら **A4縦で印刷**してください（1枚に3部分印刷されます）
4. 印刷後、横方向に切り取って3枚に分けてご利用ください

---

## 📁 ファイル構成

```
├── README.md               # この説明ファイル
└── 清水8組_回覧順序v4.docx  # 回覧順序テンプレート（名前入力欄付き）
```

---

## 📌 注意事項

- ファイルはMicrosoft Word形式（`.docx`）です
- WordがないPCの場合は [Google ドキュメント](https://docs.google.com)でも開けます
- 印刷時は **「用紙サイズ：A4」「印刷の向き：縦」** に設定してください
